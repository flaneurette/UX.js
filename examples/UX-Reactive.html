This example shows UX.JS reactivity in short:

  • UX.js uses a reactivity and state-management model similar to that of Vue.js and React.js. However, UX.js reactivity works much more simpler than React.js
  • By default UX.js calls three functions automatically: init(), render() and effect(). All other functions need to be subscribed to, by adding them into the app.functions array.
  • It is possible to call all UX.js attributes inside a module, making UX.js much more versatile.
  • app.state.mystring is the global state and can be array, object, number or string.

UX.JS reactive functions:

  • {app}.functions.{function} - subscribes a new function which can be called through data-actions.
  • {app}.state.{string} - setting of a state
  • {app}.setState(object) - setting and updating of a state with an object
  • {app}.initState(object) - initializing an object before rendering
